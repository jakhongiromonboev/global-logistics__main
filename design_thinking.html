Design Thinking Process Documentation

Solving the Traveling Salesman Problem Through Gamification

1
Empathize
User Research Insights
  • Students struggle with abstract mathematical concepts
  • Traditional TSP examples lack engagement
  • Visual learners need interactive representations
  • Gamification increases problem-solving motivation
  • Immediate feedback improves learning retention
Pain Points Identified
  • Difficulty visualizing optimization problems
  • Lack of real-world context for TSP
  • No immediate feedback on solution quality
  • Abstract nature makes learning challenging
  • Limited interactive learning tools available
Primary Persona: Alex - Computer Science Student

Age: 20-22 | Background: Undergraduate studying algorithms

Goals: Understand optimization algorithms, improve problem-solving skills, get better grades

Frustrations: Abstract concepts, lack of visual tools, difficulty connecting theory to practice

Motivations: Interactive learning, immediate feedback, competitive elements

2
Define
Problem Statement

"Computer science students need an engaging, interactive way to understand and practice the Traveling Salesman Problem because traditional learning methods fail to provide visual feedback and real-world context, leading to poor comprehension and retention of optimization concepts."

Success Metrics
  • User engagement time > 10 minutes
  • Problem completion rate > 80%
  • User satisfaction score > 4/5
  • Learning objective achievement > 90%
Design Constraints
  • Web-based for universal accessibility
  • No external dependencies required
  • Responsive design for all devices
  • Load time under 3 seconds
3
Ideate
Core Game Concepts
  • Space delivery captain theme
  • Multiple planets as destinations
  • Fuel efficiency as optimization goal
  • Real-time route visualization
  • Score-based performance feedback
Interaction Methods
  • Click-to-select planet sequence
  • Drag-and-drop route planning
  • Visual distance indicators
  • Animated path tracing
  • Comparative solution display
Visual Design Elements
  • Dark space background with stars
  • Colorful planets with unique designs
  • Glowing connection lines
  • Particle effects for engagement
  • Smooth animations and transitions
Gamification Features
  • Efficiency percentage scoring
  • Multiple difficulty levels
  • Achievement system
  • Best route comparison
  • Progress tracking
